
# 🚀 Project Ascend

> A competitive real-life ranked progression system inspired by tactical multiplayer games.

Project Ascend transforms self-improvement into a ranked competitive experience where users level up through discipline, fitness, learning, consistency, and productivity.

Instead of boring habit tracking, Ascend makes real life feel like climbing ranked.

---

# 🎯 Core Concept

Users gain **XP** and **RR (Rank Rating)** by completing real-world tasks and missions.

Failing mandatory tasks causes RR loss and possible rank demotion.

The system is designed to create:

* Progression
* Accountability
* Consistency
* Competitive motivation
* Identity-based self-improvement

---

# 🏆 Rank System

Users climb through ranks based on RR progression.

### Ranks

* Recruit
* Bronze
* Silver

* Rank progression

---

# ⚔️ Mission System

## Daily Tasks

Mandatory tasks that reward RR and XP.

Examples:

* Study sessions
* Work goals
* Fitness
* Skill development
* Health habits

Missing daily tasks causes RR loss.

---

## 🧩 Side Quests

Optional missions with higher RR rewards.

Examples:

* Extra workout
* Additional study session
* Reading
* Project building

No RR penalty for skipping side quests.

---

## 🔥 Special Quests

Rare high-value challenges with massive RR rewards.

Examples:

* Build a project
* Complete a course
* 30-day discipline challenge
* Long-term fitness goals

No RR loss for skipping special quests.

---

# 💪 Daily Discipline Routine

Core mandatory routine:

* 100 Push-Ups
* 100 Sit-Ups
* 100 Squats
* 10 KM Run

Completing the full routine grants:

* Bonus RR
* XP boosts
* Streak multipliers

Missing routine tasks causes RR penalties.
